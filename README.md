# 💬 Chat App with Socket Programming

A real-time chat application built using Python Socket Programming that allows multiple clients to communicate over a network.

## 🚀 Features
- Real-time messaging
- Multiple client support
- Private messaging
- Username management
- Thread-based communication

## 🛠️ Technologies Used
- Python
- Socket Programming
- Multithreading

## ▶️ How to Run

1. Run `server.py`
2. Run `client.py`
3. Connect multiple clients and start chatting.
